# RecordMusical

"**RecordMusical**"는 뮤지컬을 즐겨보는 사람들은 위한 어플리케이션입니다. <br/>

### 프로젝트 소개
1. RecordMusical : 뮤지컬 관람 기록을 위한 서비스<br/>
2. 기능<br/>
  2-1. 회원가입 및 로그인<br/>
  2-2. OCR 기능을 활용한 정보 입력<br/>
  2-3. Ai 챗봇을 통한 정보 검색 <br/>
  2-4. 관람 기록 기반 뮤지컬 추천, 극장별 좌석 추천 <br/>
  2-5. 네이버 지도 api를 활용한 장소 기록 기능 <br/>
  2-6. 게시판 <br/>
